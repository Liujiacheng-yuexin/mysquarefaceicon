# AGENTS.md

This is an SEO-focused web tool project for `mysquarefaceicon.com`.

Before development, SEO edits, new pages, launch checks, or content work, read:

1. `PROJECT_BRIEF.md`
2. `SEO_RULES.md`
3. `STYLE_GUIDE.md`
4. `PAGE_PLAN.md`
5. `LAUNCH_CHECKLIST.md`
6. `CONTENT_PLAN.md`

If the task involves post-launch growth, also read:

7. `SEO_PLAN.md`
8. `SEO_LOG.md`

Rules:

- Do not rely only on chat history when project documents exist.
- Do not copy competitor SWF files, images, comments, copy, or downloadable assets.
- Keep the core tool visible in the first viewport.
- Preserve SEO, performance, and mobile usability over decorative effects.
- Do not create low-quality thin pages.
- Update `SEO_LOG.md` after SEO-related production changes.

Final work summaries should include changed files, completed pages or features, SEO checks, unresolved issues, user decisions needed, and lint/build/test results.
